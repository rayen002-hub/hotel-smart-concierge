\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This graduation project presents the design and development of an intelligent digital services platform for hotel establishments, titled \textbf{Hotel Smart Concierge}. The solution aims to digitize the entire guest journey in hospitality, from digital check-in to complaint management, including bilingual messaging and information services.

The system is built as a service-oriented architecture (SOA) with four independent modules communicating via REST APIs: a React web application (PWA for the guest room area and dashboards for staff), a Node.js/Express backend with PostgreSQL database, a FastAPI artificial intelligence service (complaint classification using machine learning and multilingual translation via NLLB-200), and a Flutter mobile application for field intervention agents.

The project features a complete complaint workflow with automatic AI classification, a QR code scanning system for access control and intervention tracking, real-time bilingual messaging with automatic translation, and a housekeeping room task management module.

The Scrum methodology was adopted with 3 releases and 9 sprints. Artificial intelligence was integrated directly into the relevant functional sprints, in accordance with Scrum principles of incremental delivery.

\textbf{Keywords:} hospitality, digital services, PWA, digital check-in, complaints, AI classification, machine translation, NLLB-200, QR code, Flutter, React, Node.js, FastAPI, Scrum.
