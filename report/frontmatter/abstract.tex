\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This graduation project presents the design and development of \textbf{LoomStay}, an intelligent digital services platform for hotel establishments, carried out as part of the Professional Master's degree in Software Engineering at the Institut Supérieur d'Informatique de Monastir (ISIMM), in collaboration with \textbf{Loomens}.

The primary goal is to digitize the entire guest journey in hospitality: from secure QR-based digital check-in, to complaint management and field intervention tracking, through real-time bilingual messaging and hotel information services.

The system follows a \textbf{Service-Oriented Architecture (SOA)} pattern, composed of four independent modules communicating via REST APIs: a React web application featuring an installable PWA for the guest room space and dashboards for hotel staff; a Node.js/Express backend with a PostgreSQL database managed through Prisma ORM; a FastAPI artificial intelligence service integrating automatic complaint classification (Logistic Regression + TF-IDF) and multilingual translation via the pre-trained NLLB-200 model; and a Flutter mobile application for field intervention agents.

The main contributions of this project include a complete complaint workflow with automatic AI classification, a QR code scanning system for access control and timestamped intervention tracking, real-time bilingual guest-reception messaging with automatic translation, and a housekeeping room task management module with supervisor oversight.

The \textbf{Scrum} agile methodology was adopted, with a planning structure of \textbf{3 releases} and \textbf{9 sprints}. Artificial intelligence features were integrated directly into the relevant functional sprints, in accordance with the Scrum principle of delivering complete, potentially shippable increments.
