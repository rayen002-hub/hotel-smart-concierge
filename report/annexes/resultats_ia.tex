\subsection*{Limitations identifiées}

\begin{itemize}
  \item Messages courts ou ambigus difficiles à classifier ;
  \item Confusion entre les catégories RECEPTION et RESTAURANT ;
  \item Dataset limité ($\sim$500 exemples) — nécessité d'enrichir avec des données réelles ;
  \item La validation croisée (CV F1 macro $\sim$77\%) montre une dépendance au jeu de données.
\end{itemize}
