\chapter{Release 2 : Parcours client et services numériques}
\label{chap:release2}

\section*{Introduction}

Ce chapitre détaille la deuxième release du projet, consacrée à l'expérience digitale côté client. Cette release comprend trois sprints : l'espace chambre PWA accessible par QR, les services hôteliers (informations, événements, devises) et la messagerie bilingue en temps réel avec traduction automatique par le modèle NLLB.

\begin{quote}
\textbf{Note :} La PWA installable concerne uniquement l'espace client chambre (\texttt{/room}). La page de check-in et les dashboards du personnel restent des interfaces web classiques accessibles via navigateur.
\end{quote}

\subsection{Étude conceptuelle}

Le mécanisme d'accès repose sur :
\begin{itemize}
  \item \textbf{Token gate} : le QR code contient un token JWT signé qui est validé côté serveur (signature, dates de validité, existence de la réservation) ;
  \item \textbf{RoomLayout} : composant React protégeant toutes les routes \texttt{/room/*} ;
  \item \textbf{PWA} : configuration \texttt{vite-plugin-pwa} avec manifest et service worker, permettant l'installation sur l'écran d'accueil du smartphone.
\end{itemize}

La PWA installable concerne uniquement l'espace client chambre. Les dashboards du personnel (admin, réception, managers) restent des interfaces web classiques non installables.

Permettre la communication bilingue en temps réel entre le client et la réception, avec traduction automatique par le modèle pré-entraîné NLLB-200.

\subsection{Analyse}

\subsection{Étude conceptuelle}

Le système de messagerie bilingue repose sur :
\begin{itemize}
  \item \textbf{Modèle GuestStaffMessage} : messages bidirectionnels avec champs \texttt{originalMessage}, \texttt{detectedLanguage}, \texttt{staffMessage}, \texttt{clientMessage} ;
  \item \textbf{Service IA — Détection de langue} : bibliothèque \texttt{langdetect} avec mapping vers les codes NLLB ;
  \item \textbf{Service IA — Traduction} : modèle pré-entraîné NLLB-200-distilled-600M (Hugging Face Transformers), avec cache de traduction et lazy loading ;
  \item \textbf{Socket.IO} : communication temps réel pour les notifications de nouveaux messages.
\end{itemize}

\begin{quote}
\textbf{Note :} Le modèle NLLB-200 est un modèle pré-entraîné par Meta AI. Il n'a pas été entraîné ou fine-tuné dans le cadre de ce projet. Il est chargé et utilisé tel quel via la bibliothèque Hugging Face Transformers.
\end{quote}
